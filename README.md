# esfera-site

Base inicial do novo site institucional e acadêmico do ESFERA Aulas Particulares.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Rotas principais
- `/`
- `/ensino-fundamental`
- `/ensino-medio`
- `/ensino-superior`
- `/psicopedagogia`
- `/professores`
- `/quem-somos`
- `/contato`

## Como executar
```bash
npm install
npm run dev
```

## Estrutura-base
- `app/`: páginas e rotas com App Router
- `components/`: componentes de layout, seções, UI e formulário
- `data/`: dados mockados de níveis, disciplinas, materiais e professores
- `lib/`: tipagens e utilitários
- `styles/`: estilos globais
